# 브랜치 생성

### 1. 깃크라켓에서 바로 브랜치 생성하기

1. Branch 클릭
   1. 브랜치 이름 입력하면 생성 완료

<img width="451" alt="Untitled" src="https://github.com/inung1004/Jeong_Lee/assets/101488116/4a067954-d530-41d4-b623-6e49bb3f96d4">


### 2. 깃허브에 있는 이슈로 브랜치 생성하기

1. 레포에 들어가면 이런 화면이 뜰거에요

<img width="1440" alt="전체화면" src="https://github.com/inung1004/Jeong_Lee/assets/101488116/0beb5f86-2776-4a54-9ad8-22e9f3102fae">


2. 왼쪽 사이드바에 GITHUB ISSUES 를 누르면 그 아래에 생성한 이슈가 떠요.

   1. 만약 안 뜨면 GITHUB ISSUES 쪽에 마우스 커서를 호버하면 새로고침 버튼이 뜨는데 그 버튼을 누르면 뜰거에요.
   2. All Open Issues는 생성된 모든 이슈를 말하고, My Open Issues는 위에서 Assinees에서 자신이 추가된 이슈를 말해요.
     <img width="205" alt="Untitled" src="https://github.com/inung1004/Jeong_Lee/assets/101488116/64224cce-b495-4c1d-b340-0b60e61bd675">


3. 이제 자신에게 배정된 이슈를 누르면 이런 화면이 뜰텐데

<img width="1440" alt="Untitled" src="https://github.com/inung1004/Jeong_Lee/assets/101488116/73bfc00a-d21a-4e36-8cf9-323b696c5cd8">


4. Create a branch for this issue 버튼 클릭

   → 브랜치 이름은 맨 앞에 이슈번호 붙이고(이슈 번호는 자동으로 생성돼서 붙어있을 거에요.), Create branch 클릭

<img width="322" alt="Untitled" src="https://github.com/inung1004/Jeong_Lee/assets/101488116/06bcae8c-9d2a-48a2-9913-5967939697bc">
